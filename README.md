# Pulse Daily Summary Bot 
A Python-based automation tool that fetches daily weather/data updates using GitHub Actions.
## Features
* Automated fetching via `wttr.in` API.
* Runs seamlessly using GitHub Actions workflows.
## How It Works
The bot uses a GitHub Actions workflow (`daily.yml`) to trigger `bot.py` daily, pulling automated data updates without needing a dedicated server.
